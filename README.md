# Exercise App

## How to run the app
- Download dependencies on backend
```bash
cd [HOMEDIR]/exercise-app/backend
pnpm i
```

### Start Development Environment
1. Start the backend
  - Navigate to backend directory
  ```bash
  cd [HOMEDIR]/exercise-app/backend
  ```
  - Start the Development server
  ```bash
  pnpm dev
  ``` 

2. Start the frontend
  ```bash
  live-server
  ```
